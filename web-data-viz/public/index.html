Sobre o Projeto

O Jurassic Park: Web Genesis é um projeto desenvolvido como meio avaliativo para a matéria de Pesquisa e Inovação da SPTech.

Para além de homenagear a franquia Jurassic Park — que me é tão cara —, este projeto tem como objetivo mostrar como, através do conhecimento técnico e do uso da programação como um meio você também pode transformar suas ideias em algo concreto e significativo, se não para o mundo, então, ao menos para você!

O projeto é um jogo no estilo Park Builder. Aqui, você vai contar com diversas construções e dinossauros para gerenciar o seu próprio Jurassic Park — Sem poupar despesas!
